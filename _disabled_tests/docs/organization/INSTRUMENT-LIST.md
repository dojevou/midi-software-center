## üéØ Instrument Detection Priority

For MPC organization, instruments are detected in this order:

1. **GM Program Number** (if available from MIDI)
2. **Filename Analysis** (bass, piano, synth, etc.)
3. **Filepath Analysis** (folder names: drums, keys, etc.)
4. **Musical Analysis** (is_percussive, has_chords, has_melody)
5. **Polyphony Analysis** (monophonic vs polyphonic)
